# @comp0/react

Headless React 19 components with native-first DOM semantics, accessible interaction patterns, and no prescribed styles.

```sh
pnpm add @comp0/react react@^19 react-dom@^19
```

```tsx
import { Button } from "@comp0/react";

<Button className="button">Save</Button>;
```

Components are imported from the package root. The published JavaScript is already transformed by the React Compiler. See the [documentation](https://comp0-docs.horrible.workers.dev) for composition, styling, keyboard behavior, forms, and API details.
